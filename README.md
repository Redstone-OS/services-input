# input

Serviço de agregação de entrada.

## O que ele deve ser?
O ponto central para todos os dispositivos de entrada (teclado, mouse, touch, tablets). Ele padroniza os eventos para o sistema.

## O que precisa fazer?
- [ ] **Padronização**: Converter códigos de teclas de diferentes hardwares em um padrão Redstone.
- [ ] **Multi-device**: Suportar múltiplos ratos ou teclados simultaneamente.
- [ ] **Gestão de Layout**: Lidar com diferentes mapas de teclado (ABNT2, US, etc).
- [ ] **Filtros**: Implementar aceleração de mouse e repetição de teclas.
